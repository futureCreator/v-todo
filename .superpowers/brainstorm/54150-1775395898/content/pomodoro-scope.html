Pomodoro Timer — Where Should It Live?

뽀모도로 타이머가 앱에서 어떤 위치에 있으면 좋을까요?

A

할 일 탭 안에 통합

할 일 목록 상단에 타이머를 배치. 특정 할 일을 선택해서 뽀모도로 시작. 할 일과 시간 추적이 자연스럽게 연결됨.

B

독립된 5번째 탭

하단 네비게이션에 타이머 탭 추가. 전용 화면에서 타이머를 운영. 다른 탭을 돌아다녀도 타이머 유지.

C

플로팅 오버레이

화면 하단이나 상단에 작은 타이머 바가 항상 떠있음. 어떤 탭에서든 타이머 확인 가능. 탭하면 전체 타이머 화면 확장.
