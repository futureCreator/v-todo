# Mood Tracker (무드 트래커)

## Overview

하루 1번 기분을 이모지로 기록하고, 1년 전체를 한 페이지(Year in Pixels)로 시각화하는 기능.
노트 섹션에 통합하여 감사일기와 함께 하루 마무리 루틴으로 사용한다.

## 위치

- 노트 섹션 서브탭: **일일 / 일반 / 무드**
- 기분 입력 UI는 일일 노트 상단, 감사일기 아래에 배치

## 기분 입력

- 5단계 이모지 가로 나열: 😄 😊 😐 😔 😢
- 탭하면 즉시 저장 (POST `/api/moods`)
- 선택된 이모지는 하이라이트 표시 (배경색 + 스케일업)
- 이미 기록한 날은 선택 상태로 표시, 다시 탭하여 변경 가능
- 선택 해제는 없음 (한번 기록하면 변경만 가능)

## 연간 뷰 (무드 탭)

### 레이아웃: 세로 스트립

- 12열(1~12월) x 31행(1~31일)
- 상단: 월 라벨 (1 2 3 ... 12)
- 좌측: 일 라벨 (1 2 3 ... 31)
- 존재하지 않는 날짜(예: 2/30)는 빈 셀로 표시
- 현재 연도 고정 (연도 전환 없음)

### 셀

- 크기: 습관 히트맵과 유사한 작은 정사각형 (약 20px)
- 모서리: 약간 라운드 (2~3px)
- 오늘 날짜 셀에 테두리 강조

### 색상 매핑 (Catppuccin 테마 연동)

| 이모지 | 값 | 라이트 (Latte) | 다크 (Mocha) |
|--------|-----|----------------|--------------|
| 😄     | 5   | green          | green        |
| 😊     | 4   | teal           | teal         |
| 😐     | 3   | yellow         | yellow       |
| 😔     | 2   | peach          | peach        |
| 😢     | 1   | red            | red          |
| 미기록  | -   | surface0       | surface0     |

### 인터랙션

- 셀 탭: 토스트로 "4/8 😊" 형태의 간단한 정보 표시
- 스크롤: 31행이 화면에 안 들어갈 경우 세로 스크롤

### 범례

- 하단에 이모지 + 색상 범례 표시

## 데이터

### 저장소

- 파일: `data/moods.json`
- 형태:

```json
{
  "2026-04-08": 5,
  "2026-04-07": 3
}
```

- 키: `YYYY-MM-DD` (KST 기준 `getFullYear()/getMonth()/getDate()`)
- 값: 1~5 정수 (😢=1, 😄=5)

### API

**GET `/api/moods`**
- 쿼리: `?year=2026` (선택, 기본값 현재 연도)
- 응답: `{ "2026-04-08": 5, "2026-04-07": 3, ... }`

**POST `/api/moods`**
- 바디: `{ "date": "2026-04-08", "value": 5 }`
- 이미 존재하는 날짜면 덮어쓰기
- 응답: `{ "ok": true }`

### 스토어

- `lib/mood-store.ts`: 읽기/쓰기 함수
- atomic write 패턴 (기존 store들과 동일)

## AI 연동

### 주간 리뷰

- `lib/prompts.ts`의 주간 리뷰 프롬프트에 최근 7일 기분 데이터 추가
- AI가 기분 추이, 패턴, 감사일기/할일 완료와의 상관관계 분석

### 일일 브리핑

- 최근 7일 기분 추이를 브리핑 컨텍스트에 포함
- 기분이 3일 연속 하락 시 AI가 이를 언급
